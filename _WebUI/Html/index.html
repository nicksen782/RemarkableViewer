Sync interface:
  default: USB
Using "wifi" will use the WiFi interface on the Remarkable.
Using "usb" will use the USB interface on the Remarkable.
Open notebooks in auto-size view:
  default: YES
Setting to "YES" will fit the document images into the default size parent.
Setting to "NO" will adapt the Document View to be wider and taller based on the screen size.
addDirIdToUrl:
  default: NO
If set to "YES" then the dirId will be part of the URL and will be used on page load.
If set to "NO" then the dirId will NOT be part of the URL
addDocumentdIdToUrl:
  default: NO
If set to "YES" then the documentId will be part of the URL and will be used on page load.
If set to "NO" then the documentId will NOT be part of the URL

If set to "NO" then the pdf annotations will NOT be shown.
